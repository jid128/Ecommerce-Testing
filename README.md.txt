# E-commerce Website Testing

This project contains manual testing documentation for a demo e-commerce website.  

Files included:
- Test Plan (TestPlan.docx)
- Test Cases (TestCases.xlsx)
- Bug Report (BugReport.xlsx)
- Requirement Traceability Matrix (RTM.xlsx)

Tools Used: Jira, Excel, Chrome/Firefox
Testing Types: Functional, UI, Compatibility, Regression
